# シザリガー？それ以外？

5問の画像クイズです。ズーム画像を見て「シザリガー」か「それ以外」を判断します。

## セットアップ
- `phina.min.js` を同じフォルダに置く
- 画像は `assets/img/` に配置

## 画像ファイル名（例）
- `shizari_zoom.png`, `shizari_full.png`
- `heigani_zoom.png`, `heigani_full.png`
- `krabby_zoom.png`, `krabby_full.png`

## 公開
GitHub Pages で `index.html` をそのまま公開できます。
